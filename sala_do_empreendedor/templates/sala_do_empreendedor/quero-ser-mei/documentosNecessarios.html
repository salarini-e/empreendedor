Preparação Importante

Antes de começar o processo de formalização, veja a seguir os documentos que você precisa ter em mãos e o passo a passo deste processo. Tenha tudo organizado para uma experiência mais rápida e tranquila.

Documentos e Informações Necessárias

Prepare estes itens essenciais para completar sua formalização como MEI

Dados Pessoais

• RG (Registro Geral)
• Dados de contato atualizados
• Endereço residencial completo
• Informações básicas de identificação

Dados do Negócio

• Tipo de ocupação que exercerá
• Forma de atuação (física/online)
• Local onde o negócio será realizado
• Descrição das atividades

Conta gov.br

Acesso ao Portal de Serviços do Governo Federal através da Plataforma gov.br
• Conta ativa e senha atualizada

Se você já possui conta

A conta gov.br deve ter o nível Prata ou Ouro para brasileiros.
Verifique seu nível antes de iniciar o processo.

Se ainda não possui

Clique na opção "Fazer Cadastro" na plataforma gov.br.
Após finalizar, acesse o Portal do Empreendedor e clique em "Formalize-se".

Observações Importantes

Requisitos para conta gov.br:

  • Para Brasileiros: A conta gov.br deve ter o nível Prata ou Ouro para garantir acesso completo aos serviços de formalização.
  • Para Estrangeiros: Se a conta tiver o nível Bronze, será solicitado os dados de identificação civil do estrangeiro. Para os níveis Prata ou Ouro não será necessário informar esses dados.
    Dados adicionais necessários para estrangeiros (nível Bronze):
    • País de nacionalidade: Conforme cadastro no CPF (verifique se os dados estão atualizados na Receita Federal)
    • Documento de identificação: Emitido pela Polícia Federal. São aceitos:
      • Carteira Nacional de Registro Migratório
      • Documento Provisório de Registro Nacional Migratório
      • Protocolo de Solicitação de Refúgio
